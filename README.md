# Jom Makcik CareRide MVP

Controlled assisted mobility platform for Jom Makcik CareRide.

This repository is planned as a monorepo with one backend API, one shared database layer, separate Android app projects, and an admin web dashboard.

## Apps

- Passenger Android App: Jom Makcik CareRide
- Rider Android App: Jom Makcik Rider
- Admin Android App: Jom Makcik Admin
- Admin Web Dashboard: Jom Makcik Control Centre

## Monorepo Structure

- `backend/` - Backend API and local MVP database integration
- `passenger-app/` - Passenger Android app
- `rider-app/` - Rider Android app
- `admin-app/` - Admin Android app
- `admin-web/` - Admin web dashboard
- `shared/` - Shared constants, validation rules, and types
- `docs/` - MVP notes, safety rules, and phase documentation

## Current Status

Phase 0 is in progress. No runnable app or backend has been created yet.
