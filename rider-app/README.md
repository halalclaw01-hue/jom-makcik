# Jom Makcik Rider

Expo React Native foundation for the Rider Android app.

## Local setup

```bash
npm install
npm run android
```

The default API base URL is `http://10.0.2.2:4000`, which points an Android emulator to the host backend.

## Current Phase

Phase 23 creates the rider app shell, screen navigation, API client, token storage, and real backend login/register calls only.
