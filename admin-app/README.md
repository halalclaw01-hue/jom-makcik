# Jom Makcik Admin

Light Expo React Native admin app for urgent monitoring.

## Local setup

```bash
npm install
npm run android
```

The default API base URL is `http://10.0.2.2:4000`, which points an Android emulator to the host backend.

## Current Phase

Phase 26 creates the mobile admin shell, screen navigation, API client, token storage, and real backend admin login only.
